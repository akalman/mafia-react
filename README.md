# mafia-react

## Install Instructions

1. install nodejs
1. install global dependencies
   
   ```
   npm install -g yarn typescript
   ```
1. clone repository
   
   ```
   git clone git@github.com:akalman/mafia-react
   ```

## Build Instructions

1. install dependencies
   
   ```
   yarn
   ```
1. build
   
   ```
   yarn build
   ```
1. start server
   
   ```
   yarn start
   ```
